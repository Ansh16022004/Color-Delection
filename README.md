# Color-Delection
Color Detection is a computer vision technique used to identify and classify colors in an image or real-time video. 
It works by reading pixel values (RGB/HSV) and detecting which color range they belong to. This system is used 
in robotics, image processing, object tracking, and automation 
Tools & Technologies 
• Language: Python 
• Libraries: OpenCV, NumPy 
• Tools: VS Code / Jupiter Notebook
